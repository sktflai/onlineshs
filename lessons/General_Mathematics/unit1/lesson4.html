Lesson 4: Inverse Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Determine if a function is one-to-one using the horizontal line test
  • Find the inverse of a function algebraically
  • Graph the inverse by reflecting over y = x
  • Verify inverses using composition
  • Understand domain and range swap for inverses

One-to-One Functions

A function is one-to-one if it passes the horizontal line test (no horizontal line intersects the graph more than once).

Linear functions (slope ≠ 0) are one-to-one.
Quadratic functions are not (parabola fails the test).

Finding Inverse Algebraically

Swap x and y, then solve for y.

Example 1: f(x) = 3x + 6
y = 3x + 6 → x = 3y + 6 → y = (x - 6)/3
f⁻¹(x) = (x - 6)/3
Example 2: f(x) = √(x + 5), x ≥ -5
y = √(x + 5) → y² = x + 5 → x = y² - 5 (y ≥ 0)
f⁻¹(x) = x² - 5, x ≥ 0
Example 3 (Temperature): F to C: C(F) = (5/9)(F - 32)
Inverse C to F: F(C) = (9/5)C + 32

Graphing Inverse Functions

Reflect the graph of f over the line y = x.

Verifying Inverses

(f ∘ f⁻¹)(x) = x and (f⁻¹ ∘ f)(x) = x

For f(x) = 3x + 6, f⁻¹(x) = (x - 6)/3
(f ∘ f⁻¹)(x) = 3*((x - 6)/3) + 6 = x

Domain and Range of Inverse

They swap: Domain of f⁻¹ = Range of f, Range of f⁻¹ = Domain of f

Practice Exercises

Test your understanding.

  1. Find inverse of f(x) = 4x - 8.
  2. Is f(x) = x² one-to-one? Restrict domain to make it so.
  3. Verify f(x) = x + 5 and f⁻¹(x) = x - 5 are inverses.
  4. Real-life: Speed s = distance d / time t. Find inverse for t.
Answers:

1. f⁻¹(x) = (x + 8)/4
2. No. Restrict to x ≥ 0 or x ≤ 0
3. (f ∘ f⁻¹)(x) = x ✓
4. t = d / s
