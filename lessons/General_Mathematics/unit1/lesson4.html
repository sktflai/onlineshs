Lesson 4: Inverse Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Determine if a function is one-to-one using the horizontal line test
  • Find the inverse of a function algebraically
  • Graph the inverse by reflecting over y = x
  • Verify inverses using composition
  • Understand domain and range swap for inverses

One-to-One Functions

A function is one-to-one if every output comes from exactly one input (passes horizontal line test).

Linear functions (non-horizontal) are one-to-one.
Quadratic functions are not (parabola fails horizontal test).

Finding Inverse Algebraically

Steps: Swap x and y, solve for y.

Example 1 (Linear): f(x) = 3x + 6
y = 3x + 6 → x = 3y + 6 → x - 6 = 3y → y = (x - 6)/3
f⁻¹(x) = (x - 6)/3
Example 2 (Square Root): f(x) = √(x + 5), x ≥ -5
y = √(x + 5) → y² = x + 5 → x = y² - 5, y ≥ 0
f⁻¹(x) = x² - 5, x ≥ 0
Example 3 (Real-life Temperature): C to F: f(C) = (9/5)C + 32
Inverse F to C: f⁻¹(F) = (5/9)(F - 32)

Graphing Inverse Functions

The graph of f⁻¹ is the reflection of f over the line y = x.

Verifying Inverses

(f ∘ f⁻¹)(x) = x and (f⁻¹ ∘ f)(x) = x

Using Example 1: f(x) = 3x + 6, f⁻¹(x) = (x - 6)/3
(f ∘ f⁻¹)(x) = 3*((x - 6)/3) + 6 = x - 6 + 6 = x ✓

Domain and Range of Inverse

Domain of f⁻¹ = Range of f
Range of f⁻¹ = Domain of f

Practice Exercises

Test your understanding.

  1. Find inverse of f(x) = 4x - 8.
  2. Is f(x) = x² one-to-one? Why? Restrict domain to make it one-to-one.
  3. Verify f(x) = x + 5, f⁻¹(x) = x - 5 are inverses using composition.
  4. Real-life: Distance d = speed × time. Find inverse for time in terms of d and s.
Answers:

1. f⁻¹(x) = (x + 8)/4
2. No (fails horizontal test). Restrict x ≥ 0 or x ≤ 0
3. (f ∘ f⁻¹)(x) = (x - 5) + 5 = x ✓
4. t = d / s
