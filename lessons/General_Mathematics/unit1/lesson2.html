Practice Exercises

Apply what you've learned.

  1. Create a table of values for f(x) = x² - 1 using x = -2 to 2.
  2. Describe verbally: A taxi fare is ₱40 flag down + ₱13.50 per km.
  3. Write the equation for: "Salary is ₱15,000 base + ₱500 per item sold."
  4. Which representation is best for predicting future values? Why?
Answers:

1. x: -2→3, -1→0, 0→-1, 1→0, 2→3
2. The total fare is 40 pesos plus 13.50 pesos for each kilometer traveled.
3. S(n) = 15000 + 500n (n = items sold)
4. Equation — easiest for calculation and prediction
