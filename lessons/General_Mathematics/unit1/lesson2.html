Lesson 2: Representations of Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Represent functions using tables, graphs, equations, mappings, and verbal descriptions
  • Interpret real-life situations as functions in different forms
  • Convert between different representations of the same function
  • Graph linear and quadratic functions from tables or equations

Why Different Representations?

Each form helps us understand functions better in different contexts:

  • Table: Shows exact values
  • Graph: Shows trends and behavior
  • Equation: Allows calculations and predictions
  • Verbal: Describes real-world meaning
  • Mapping: Shows input-output pairs clearly

1. Table of Values

List input (x) and corresponding output f(x).

Example: f(x) = 2x + 1

2. Graph

Plot points from table and connect (for continuous functions).

Graphing Steps:
1. Make a table of values
2. Plot points (x, y)
3. Connect with a smooth line or curve

3. Equation (Rule)

Mathematical formula: f(x) = mx + b (linear), f(x) = ax² + bx + c (quadratic)

The cost C of printing n flyers is C(n) = 500 + 20n
Fixed cost 500 + variable 20 per flyer

4. Verbal Description

Words describing the relationship.

"The total cost is 500 pesos plus 20 pesos for each flyer printed."
→ This describes the same function as above.

5. Mapping Diagram

Arrows show how inputs map to outputs.

Real-Life Modeling Examples

Distance-Time (Motion):
A car travels at constant 60 km/h.
Distance d(t) = 60t (t in hours)
Table: t=0 → 0km, t=1 → 60km, t=2 → 120km
Graph: Straight line through origin
Cost of Rice:
Price per kilo = ₱50
Total cost C(w) = 50w (w = weight in kg)
Constant rate → linear function

Practice Exercises

Apply what you've learned.

  1. Create a table of values for f(x) = x² - 1 using x = -2 to 2.
  2. Describe verbally: A taxi fare is ₱40 flag down + ₱13.50 per km.
  3. Write the equation for: "Salary is ₱15,000 base + ₱500 per item sold."
  4. Which representation is best for predicting future values? Why?
Answers:

1. x: -2→3, -1→0, 0→-1, 1→0, 2→3
2. The total fare is 40 pesos plus 13.50 pesos for each kilometer traveled.
3. S(n) = 15000 + 500n (n = items sold)
4. Equation — easiest for calculation and prediction
