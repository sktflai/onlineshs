Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Distinguish between relations and functions
  • Represent functions using ordered pairs, tables, mappings, graphs, and equations
  • Use the vertical line test to determine if a graph represents a function
  • Identify the domain and range of a function
  • Evaluate functions using function notation
  • Recognize common types of functions and their basic graphs

What is a Function?

A relation is any set of ordered pairs (x, y).

A function is a special type of relation where each input (x-value, or domain element) has exactly one output (y-value, or range element). No x-value repeats with different y-values.

In real life: Think of a function like a vending machine — you press one button (input), you get exactly one snack (output).

Example 1: Which are functions?
A: {(1,2), (2,3), (3,4)} → Yes (function)
B: {(1,2), (1,3), (2,4)} → No (1 maps to two values)

Representations of Functions

Functions can be shown in many ways:

1. Ordered Pairs

{(1,3), (2,5), (3,7)}

2. Table of Values

xy = 2x + 1
-1-1
01
13

3. Graph

Use the vertical line test: If any vertical line touches the graph more than once, it's not a function.

4. Equation

f(x) = 2x + 1 (function notation)

Domain and Range

Domain: All possible x-values (inputs)
Range: All possible y-values (outputs)

Example: For f(x) = x²
Domain: All real numbers (-∞, ∞)
Range: y ≥ 0 [0, ∞)

Function Notation and Evaluation

f(x) means "output when input is x"

Example: If f(x) = 3x - 2
f(4) = 3(4) - 2 = 10
f(-1) = 3(-1) - 2 = -5

Common Types of Functions

  • Linear: f(x) = mx + b → straight line
  • Quadratic: f(x) = ax² + bx + c → parabola
  • Constant: f(x) = c → horizontal line
