Relation vs Function

A relation is any set of ordered pairs.

A function is a relation where each input (x) has exactly one output (y).

Real-life example: A student's ID number (input) → exactly one name (output) = function.
But one name → multiple ID numbers = not a function.

Example:
Relation A: {(1,4), (2,5), (3,6)} → Function
Relation B: {(1,4), (1,7), (2,5)} → Not a function (1 has two outputs)

Domain and Range

Domain: All possible x-values (inputs)
Range: All possible y-values (outputs)

Example 1: f(x) = x + 3
Domain: All real numbers (-∞, ∞)
Range: All real numbers (-∞, ∞)
Example 2: f(x) = √x
Domain: x ≥ 0 [0, ∞)
Range: y ≥ 0 [0, ∞)

Function Notation & Evaluation

f(x) = "f of x" = output when input is x

If f(x) = 2x - 5,
f(3) = 2(3) - 5 = 1
f(-1) = 2(-1) - 5 = -7
f(a) = 2a - 5

Common Types of Functions & Graphs

  • Constant: f(x) = c → horizontal line
  • Linear: f(x) = mx + b → straight line
  • Quadratic: f(x) = ax² + bx + c → parabola
