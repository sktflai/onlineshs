Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Recall the difference between a relation and a function
  • Represent functions in various ways (ordered pairs, table, mapping, graph, equation)
  • Use the vertical line test to identify functions
  • Determine the domain and range of a function
  • Evaluate functions using function notation f(x)
  • Identify common types of functions (constant, linear, quadratic, etc.)

Relation vs Function

A relation is any set of ordered pairs.

A function is a relation where each input (x) has exactly one output (y).

Real-life example: Student ID → Name = function. But Name → multiple IDs = not a function.

Example:
{(1,4), (2,5), (3,6)} → Function
{(1,4), (1,7), (2,5)} → Not a function

Representations of Functions

Functions can be represented as:

  • Ordered pairs
  • Table of values
  • Mapping diagram
  • Graph
  • Equation

Vertical Line Test

If a vertical line intersects the graph more than once → not a function.

Domain and Range

Domain: Set of all possible x-values
Range: Set of all possible y-values

Example 1: f(x) = 2x + 1
Domain: All real numbers
Range: All real numbers
Example 2: f(x) = √x
Domain: x ≥ 0
Range: y ≥ 0

Function Notation & Evaluation

f(x) reads "f of x"

If f(x) = 3x - 2,
f(5) = 3(5) - 2 = 13
f(-1) = 3(-1) - 2 = -5

Common Types of Functions

  • Constant: horizontal line
  • Linear: straight line
  • Quadratic: parabola

Practice Exercises

Test your understanding before checking answers.

  1. Determine whether {(3,5), (4,6), (3,7), (5,8)} is a function.
  2. If f(x) = 2x + 3, find f(0), f(5), f(-2), and f(a + 1).
  3. State the domain and range of f(x) = √(x - 1).
  4. Is a circle a function? Explain using the vertical line test.
  5. Identify the type: f(x) = 7 (constant or linear?)
Answers:

1. Not a function (x=3 has two outputs)
2. f(0)=3, f(5)=13, f(-2)=-1, f(a+1)=2a+5
3. Domain: x ≥ 1, Range: y ≥ 0
4. No — vertical line intersects at two points
5. Constant function
