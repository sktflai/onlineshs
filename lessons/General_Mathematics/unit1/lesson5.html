Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Understand and graph the absolute value function
  • Graph and evaluate floor and ceiling functions
  • Recognize step functions and their real-life uses
  • Understand the signum function
  • Define, graph, and evaluate piecewise functions for real-life modeling

Absolute Value Function

f(x) = |x| — the distance from x to 0, always non-negative. Graph is V-shaped.

f(x) = |x - 2| + 1 — shifted right 2, up 1

Greatest Integer (Floor) and Ceiling Functions

Floor ⌊x⌋ = greatest integer ≤ x (rounds down)
Ceiling ⌈x⌉ = least integer ≥ x (rounds up)

⌊3.8⌋ = 3, ⌊-1.2⌋ = -2
⌈3.8⌉ = 4, ⌈-1.2⌉ = -1

Step Functions

Constant over intervals, jumps at integers — used for tiered pricing.

Shipping cost: ₱100 for 0-5kg, ₱200 for 5-10kg, etc.

Signum Function

sgn(x) = -1 (x < 0), 0 (x = 0), 1 (x > 0)

sgn(-5) = -1, sgn(0) = 0, sgn(7) = 1

Piecewise Functions

Different formulas for different intervals.

Phone plan: f(m) = 500 if m ≤ 100, 500 + 10(m - 100) if m > 100

Practice Exercises

Apply the concepts.

  1. Evaluate | -4 | and | 3 - 7 |.
  2. ⌊5.9⌋, ⌈2.1⌉, ⌊-0.5⌋
  3. sgn(-10), sgn(0), sgn(4)
  4. Piecewise: f(x) = x if x < 0, 0 if x = 0, -x if x > 0 (absolute value!). Evaluate at -3, 0, 5.
Answers:

1. 4, 4
2. 5, 3, -1
3. -1, 0, 1
4. 3, 0, 5
