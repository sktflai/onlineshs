Lesson 5: Piecewise & Special Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Graph and evaluate the absolute value function
  • Understand greatest integer (floor) and least integer (ceiling) functions
  • Work with step functions and signum function
  • Define, graph, and evaluate piecewise functions
  • Model real-life situations using piecewise functions

1. Absolute Value Function

f(x) = |x| — V-shaped graph, vertex at (0,0)

f(x) = |x - 2| + 3 → shifted right 2, up 3

2. Greatest Integer (Floor) & Least Integer (Ceiling)

Floor ⌊x⌋ = greatest integer ≤ x
Ceiling ⌈x⌉ = least integer ≥ x

⌊3.7⌋ = 3, ⌊-1.2⌋ = -2
⌈3.7⌉ = 4, ⌈-1.2⌉ = -1

3. Step Functions

Constant over intervals — "stairs"

4. Signum Function

sgn(x) = -1 if x < 0, 0 if x = 0, 1 if x > 0

5. Piecewise Functions

Different rule for different intervals

0.20(x - 250000) if 250000 < x ≤ 500000

Practice Exercises

Apply the concepts.

  1. Evaluate f(x) = |x - 3| at x = 5, x = 1, x = 3.
  2. ⌊4.8⌋, ⌈-2.3⌉, ⌊-2.3⌋
  3. Piecewise: f(x) = x if x < 0, x² if x ≥ 0. Find f(-2), f(3).
  4. Real-life: Parking ₱40 first hour, ₱20 each additional. Write piecewise for 3.5 hours.
Answers:

1. 2, 2, 0
2. 4, -2, -3
3. -2, 9
4. ₱40 + 2×₱20 = ₱80 (or piecewise definition)
