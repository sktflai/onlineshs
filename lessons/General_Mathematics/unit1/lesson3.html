Lesson 3: Operations on Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Perform addition, subtraction, multiplication, and division on functions
  • Find the domain of the resulting function
  • Perform composition of functions (f ∘ g)(x)
  • Solve real-life problems involving operations on functions

Arithmetic Operations on Functions

The domain of the result is the intersection of the domains of f and g (except for division, where g(x) ≠ 0).

Example 1: f(x) = 3x - 1, g(x) = x² + 2
(f + g)(x) = x² + 3x + 1
(f - g)(x) = -x² + 3x - 3
Example 2 (Multiplication): f(x) = x + 5, g(x) = x - 4
(f · g)(x) = x² + x - 20
Example 3 (Division): f(x) = 4x, g(x) = x + 1
(f / g)(x) = 4x / (x + 1), x ≠ -1

Composition of Functions

(f ∘ g)(x) = f(g(x)) — apply g first, then f.

Example 1: f(x) = √x, g(x) = x + 4
(f ∘ g)(x) = √(x + 4), domain x ≥ -4
(g ∘ f)(x) = √x + 4, domain x ≥ 0
Example 2 (Real-life Temperature): F to C: C(x) = (5/9)(x - 32)
C to F: F(y) = (9/5)y + 32
(F ∘ C)(x) = x — they cancel out!

Real-Life Application: Profit = Revenue - Cost

Revenue R(x) = selling price × quantity
Cost C(x) = fixed + variable × quantity
Profit P(x) = R(x) - C(x)

Practice Exercises

Try these varied problems.

  1. f(x) = x + 4, g(x) = 2x - 1. Find (f + g)(x) and (f · g)(x).
  2. Find (f ∘ g)(x) and domain if f(x) = 1/x, g(x) = x - 3.
  3. Revenue R(x) = 80x, Cost C(x) = 20x + 2000. Find profit P(x).
Answers:

1. (f + g)(x) = 3x + 3, (f · g)(x) = 2x² + 7x - 4
2. (f ∘ g)(x) = 1/(x - 3), domain x ≠ 3
3. P(x) = 60x - 2000
