Lesson 3: Operations on Functions

Unit 1: Functions and Their Graphs | General Mathematics - Grade 11 STEM

Learning Objectives

  • Perform addition, subtraction, multiplication, and division on functions
  • Find the domain of the resulting function
  • Perform composition of functions (f ∘ g)(x)
  • Solve real-life problems involving operations on functions

Arithmetic Operations on Functions

For functions f(x) and g(x), with domain intersection D:

  • (f + g)(x) = f(x) + g(x)
  • (f - g)(x) = f(x) - g(x)
  • (f · g)(x) = f(x) · g(x)
  • (f / g)(x) = f(x) / g(x), g(x) ≠ 0

Examples of Arithmetic Operations

Let f(x) = 2x + 1 and g(x) = x² - 3

(f + g)(x) = (2x + 1) + (x² - 3) = x² + 2x - 2
(f - g)(x) = (2x + 1) - (x² - 3) = -x² + 2x + 4
(f · g)(x) = (2x + 1)(x² - 3) = 2x³ - 6x + x² - 3 = 2x³ + x² - 6x - 3
(f / g)(x) = (2x + 1)/(x² - 3), x² ≠ 3

Composition of Functions

Composition is "function of a function": (f ∘ g)(x) = f(g(x))
Order matters: (f ∘ g) ≠ (g ∘ f) generally

Let f(x) = 3x - 2 and g(x) = x²

(f ∘ g)(x) = f(g(x)) = 3(x²) - 2 = 3x² - 2
(g ∘ f)(x) = g(f(x)) = (3x - 2)² = 9x² - 12x + 4

Real-Life Application

Cost example: Fixed cost f(x) = 5000, variable cost g(x) = 20x (x = units produced)
Total cost C(x) = f(x) + g(x) = 5000 + 20x

Practice Exercises

Test your skills.

  1. Let f(x) = x + 4, g(x) = 2x - 1. Find (f + g)(x) and (f - g)(x).
  2. Find (f · g)(x) for the same functions.
  3. Find (f ∘ g)(x) and (g ∘ f)(x).
  4. Real-life: Revenue R(x) = 50x, Cost C(x) = 20x + 1000. Find profit P(x) = R(x) - C(x).
Answers:

1. (f + g)(x) = 3x + 3, (f - g)(x) = -x + 5
2. (f · g)(x) = 2x² + 6x - 4
3. (f ∘ g)(x) = 2x + 3, (g ∘ f)(x) = 2x + 7
4. P(x) = 30x - 1000
