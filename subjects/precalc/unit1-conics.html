Unit 1: Analytic Geometry – Conic Sections

Grade 11 STEM – DepEd Philippines Curriculum

Learning Objectives:

  • Illustrate and identify the different conic sections (circle, parabola, ellipse, hyperbola)
  • Determine the standard form of the equation of each conic section
  • Graph each conic section and identify its key parts (center, vertices, foci, etc.)
  • Solve situational problems involving conic sections

Introduction to Conic Sections

Conic sections are curves obtained by intersecting a plane with a double-napped cone. Depending on the angle of the plane, we get a circle, parabola, ellipse, or hyperbola.

1. Circle

Definition: The set of all points equidistant from a fixed point called the center.

Standard Equation: (x − h)² + (y − k)² = r²

Center: (h, k)    Radius: r

Example 1: Find the center and radius.
(x + 5)² + (y − 2)² = 49
→ Center: (−5, 2), Radius: 7

Example 2: Write the equation of a circle with center (3, −4) and radius 8.
→ (x − 3)² + (y + 4)² = 64

2. Parabola

Definition: The set of all points equidistant from a fixed point (focus) and a fixed line (directrix).

Vertical: (x − h)² = 4p(y − k)    (opens up or down)

Horizontal: (y − k)² = 4p(x − h)    (opens left or right)

Vertex: (h, k)    Focus: p units from vertex along axis of symmetry

Example: Identify vertex, focus, and direction.
(x − 1)² = −8(y + 3)
→ Vertex: (1, −3), p = −2, opens downward, Focus: (1, −5)

Helpful Video: Parabolas – The Organic Chemistry Tutor

3. Ellipse

Definition: The set of all points where the sum of distances to two fixed points (foci) is constant.

Horizontal major axis: \frac{(x - h)^2}{a^2} + \frac{(y - k)^2}{b^2} = 1    (a > b)

c² = a² − b²    Foci: (±c, 0) from center

Example: \frac{x^2}{25} + \frac{y^2}{9} = 1
a = 5, b = 3, c = 4, Foci: (±4, 0)

Helpful Video: Khan Academy – Ellipse Features

4. Hyperbola

Definition: The set of all points where the difference of distances to two fixed points (foci) is constant.

Horizontal: \frac{(x - h)^2}{a^2} - \frac{(y - k)^2}{b^2} = 1

c² = a² + b²    Asymptotes: y − k = ±(b/a)(x − h)

Example: \frac{x^2}{16} - \frac{y^2}{9} = 1
a = 4, b = 3, c = 5, Asymptotes: y = ±(3/4)x
