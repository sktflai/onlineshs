Unit 3: Trigonometric Functions, Identities, and Equations

Grade 11 STEM – DepEd Philippines Curriculum

Learning Objectives:

  • Understand angles in radians and degrees using the unit circle
  • Define, evaluate, and graph the six trigonometric functions
  • Prove and apply fundamental trigonometric identities
  • Work with inverse trigonometric functions
  • Solve basic trigonometric equations

1. Angles and the Unit Circle

Angles can be measured in degrees (360° = full circle) or radians (2π radians = full circle). The unit circle is a circle with radius 1 centered at the origin, where any point on the circle is (cos θ, sin θ).

2. The Six Trigonometric Functions

sin θ = opposite/hypotenuse, cos θ = adjacent/hypotenuse, tan θ = opposite/adjacent
csc θ = 1/sin θ, sec θ = 1/cos θ, cot θ = 1/tan θ

Periods: sin/cos/csc/sec = 2π, tan/cot = π

Example Values: sin(π/6) = 1/2, cos(π/3) = 1/2, tan(π/4) = 1

Helpful Video: Graphing Trig Functions - Khan Academy

3. Fundamental Trigonometric Identities

These are equations true for all values where defined:

  • Pythagorean: sin²θ + cos²θ = 1, 1 + tan²θ = sec²θ, 1 + cot²θ = csc²θ
  • Reciprocal: csc θ = 1/sin θ, etc.
  • Quotient: tan θ = sin θ / cos θ
  • Co-function: sin(π/2 - θ) = cos θ

Example Proof: sin²θ + cos²θ = 1 comes from Pythagorean theorem on the unit circle (x² + y² = 1).

4. Inverse Trigonometric Functions

arcsin(x), arccos(x), arctan(x) return the angle whose trig function equals x.

Ranges: arcsin → [-π/2, π/2], arccos → [0, π], arctan → (-π/2, π/2)

Example: arcsin(√3/2) = π/3, arctan(1) = π/4

Helpful Video: Inverse Trig Functions - Khan Academy

5. Solving Trigonometric Equations

Solve by isolating the trig function and finding all solutions in the interval, considering periodicity.

Example: Solve 2 cos θ + 1 = 0 for 0 ≤ θ < 2π
cos θ = -1/2 → θ = 2π/3, 4π/3
